Expense Manager App

Description:
Developed a cross-platform Expense Manager mobile application using Flutter with Provider for state management and Firebase as the backend. The app enables users to efficiently track daily expenses and income, categorize transactions, visualize spending patterns through charts, and maintain a secure, cloud-synced financial record. Integrated Firebase Authentication for user login/signup and Firestore for real-time data storage, ensuring seamless synchronization across devices.

Key Features:
Add, edit, and delete income and expense transactions.

Categorize expenses with icons and color codes.

Visual analytics with pie and bar charts for financial insights.

User authentication (email/password via Firebase Auth).

Real-time cloud storage with Cloud Firestore.

Responsive UI with Provider for efficient state management.

Technologies Used:

Frontend: Flutter, Dart

State Management: Provider

Backend: Firebase Authentication, Cloud Firestore

Tools: Android Studio, Firebase Console, VS Code
